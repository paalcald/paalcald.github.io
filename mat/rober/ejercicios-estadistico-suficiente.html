Ejercicios de Estadísticos Suficientes

Encuentresé un estadístico suficiente para una m.a.s. de tamaño n de cada una de las siguientes poblaciones

\( X \sim Beta(\alpha, \beta) \quad \alpha > 0, \quad \beta > 0 \)

Por ser común a todos los casos, en primer lugar dado que la función de densidad de probabilidad asociada es \( f_{\alpha,\beta} \left( x \right) = \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right) \Gamma \left( \beta \right)}x^{\alpha-1} \cdot \left( 1 - x \right)^{\beta-1} \) calculamos la de n sucesos independientes como su producto.

\begin{align*} f_{\alpha,\beta} \left( \vec{x} \right) &= \prod_{i = 1}^n\left( \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right) \Gamma \left( \beta \right)}x^{\alpha-1} \cdot \left( 1 - x \right)^{\beta-1} \right) \\ &= \left( \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right) \Gamma \left( \beta \right)} \right)^n \prod_{i = 1}^n\left( x^{\alpha-1} \cdot \left( 1 - x \right)^{\beta-1} \right) \\ &= \left( \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right) \Gamma \left( \beta \right)} \right)^n \left( \prod_{i = 1}^n x_i \right)^{\alpha - 1} \left( \prod_{i = 1}^n (1 - x_i) \right)^{\beta - 1} \end{align*}

α desconocido y β conocido

Aplicando el Teorema de Factorización de Fisher-Neyman podemos descomponer \(f_{\alpha,\beta} \left( \vec{x} \right) \) de la siguiente manera.

\begin{gather*} h \left( \vec{x} \right) = \frac{1}{\Gamma \left( \beta \right)^n} \left( \prod_{i = 1}^n (1 - x_i) \right)^{\beta - 1} \\ \left( g_{\alpha} \circ T \right) \left( \vec{x} \right)= \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right)} \right)^n \left( \prod_{i = 1}^n x_i \right)^{\alpha - 1} \end{gather*}

De donde podemos concluir que \( T(\vec{x}) = \prod_{i = 1}^n x_i \) es suficiente para α.

α conocido y β desconocido

Aplicando el Teorema de Factorización de Fisher-Neyman podemos descomponer \(f_{\alpha,\beta} \left( \vec{x} \right) \) de la siguiente manera.

\begin{gather*} h \left( \vec{x} \right) = \frac{1}{\Gamma \left( \alpha \right)^n} \left( \prod_{i = 1}^n x_i \right)^{\alpha - 1} \\ \left( g_{\beta} \circ T \right) \left( \vec{x} \right) = \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \beta \right)} \right)^n \left( \prod_{i = 1}^n (1 - x_i) \right)^{\beta - 1} \end{gather*}

De donde podemos concluir que \( T(\vec{x}) = \prod_{i = 1}^n (1 - x_i) \) es suficiente para β.

α desconocido y β desconocido

Recordemos que un estadístico puede ser de la forma \( T:\mathbb{R}^n \to \mathbb{R}^m \), en este caso tomaremos \( T:\mathbb{R}^n \to \mathbb{R}^2 \) ya que necesitaremos estimar dos parámetros.

\begin{gather*} h \left( \vec{x} \right) = 1 \\ \left( g_{\alpha,\beta} \circ T \right) \left( \vec{x} \right) = \left( \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right) \Gamma \left( \beta \right)} \right)^n \left( \prod_{i = 1}^n x_i \right)^{\alpha - 1} \left( \prod_{i = 1}^n (1 - x_i) \right)^{\beta - 1} \end{gather*}

De donde podemos concluir que \( T(\vec{x}) = \left( T_1 \left( \vec{x} \right), T_2 \left( \vec{x} \right) \right) = \left( \prod_{i = 1}^n x_i, \prod_{i = 1}^n (1 - x_i) \right) \) es suficiente para α, β.

Observación, nótese que explícitamente la función sería \( g_{\alpha,\beta} \left( t_1, t_2 \right) = \left( \frac{\Gamma \left( \alpha + \beta \right)}{ \Gamma \left( \alpha \right) \Gamma \left( \beta \right)} \right)^n \left( t_1 \right)^{\alpha - 1} \left( t_2 \right)^{\beta - 1} \)
