Teorema Central de Límite

Si \(E[X] < \infty,\quad V[X] < \infty\) entonces \(n \gg 0 \quad \frac{X-E[X]}{\sqrt{V[X]}} \stackrel{\mathcal{L}}{\longrightarrow} N(0,1)\)

Normal es Reproductiva

\(\begin{gathered}X \sim N(\mu_X, \sigma^2_X) \\ Y \sim N(\mu_Y, \sigma^2_Y)\end{gathered} \quad X + Y \sim N(\mu_X + \mu_Y, \sigma^2_X + \sigma^2_Y)\)

\(\overline{X} = \frac{\sum x_i}{n}\) es la manera más natural de agrupar datos.

\(E[\overline{X}] = \text{¿?}\)

\(V[\overline{X}] = \text{¿?}\)

\(\{X_n| n \ge 1 \}\) v.a.i.i.d.

\(E[\overline{X}] = E \left[ \frac{\sum x_i}{n} \right] = \frac{1}{n} E[\sum x_i] = \frac{1}{n} \sum E[x_i] \stackrel{\text{i.d.}}{=} \frac{1}{n} n E[X] = E[X]\)

\(V[\overline{X}] = V \left[ \frac{\sum x_i}{n} \right] = \frac{1}{n^2}V \left[ \sum x_i \right] \stackrel{\text{v.a.i.}}{= } \frac{\sum V[x_i]}{n^2} \stackrel{\text{i.d.}}{= } \frac{nV[X]}{n^2} = \frac{V[X]}{n}\)

Teorema de Fischer

Sea \(\{ X_n \ N \ge 1 \}\) v.a.i.i.d. \(X \sim N(\mu, \sigma^2)\)

  1. \(\overline{X}\) y \(S^2_n\) son independientes
  2. \(\frac{n S^2_n}{\sigma^2} \sim \chi^2_n\)

Observación \(\chi^2\) es reproductiva entonces \(\chi^2_n + \chi^2_m = \chi^2_{n+m}\) , \(E[\chi^2_n] = n\) , \(V[\chi^2_n] = 2n\)

Distribución T-student de n grados de libertad

\(T_n = \frac{N(0,1)}{\sqrt{\frac{\chi^{2_n}}{n}}}\)

\(F_{n,m} = \frac{\frac{\chi^2_n}{n}}{\frac{\chi^2_m}{m}}\)

Sean \((X_{1}, \dots, X_{13})\) m.a.s. n=13 con \(X_i \sim N(0,1)\) v.a.i.i.d.

\(X = \frac{X_1 + X_2 + X_3}{\sqrt{(X_4 + X_5)^2 + (X_6 + X_7)^2 + (X_8 + X_9)^2 + (X_{10} + X_{11})^2 + (X_{12} + X_{13})^2}}\) calcular el valor de k para el que \(P(X < k) = 0.99\)

\(X_1 + X_2 + X_3 \sim N(0, \sigma^2 = 3)\) por lo tanto tipifico esta parte

\(X = \sqrt{3}\frac{\frac{X_1 + X_2 + X_3}{\sqrt{3}}}{\sqrt{(X_4 + X_5)^2 + (X_6 + X_7)^2 + (X_8 + X_9)^2 + (X_{10} + X_{11})^2 + (X_{12} + X_{13})^2}}\)

\(X_i + X_j \sim N(0, \sigma^{2} = 2)\) por lo tanto también puedo tipificar esto!

\(\frac{\sqrt{2}}{\sqrt{3}}X = \frac{\frac{X_1 + X_2 + X_3}{\sqrt{3}}}{\sqrt{\left(\frac{X_4 + X_5}{\sqrt{2}}\right)^2 + \left(\frac{X_6 + X_7}{\sqrt{2}}\right)^2 + \left(\frac{X_8 + X_9}{\sqrt{2}}\right)^2 + \left(\frac{X_{10} + X_{11}}{\sqrt{2}}\right)^2 + \left(\frac{X_{12} + X_{13}}{\sqrt{2}}\right)^2}}\)

Y ahora identificamos el denominador como \(\chi^2_5\) así que

\(\sqrt{5}\frac{\sqrt{2}}{\sqrt{3}}X = \frac{N(0,1)}{\sqrt{\frac{\chi^{2}_5}{5}}}\) de donde concluimos que \(\frac{\sqrt{5}\sqrt{2}}{\sqrt{3}}X = \text{T}_5\) y podemos calcular el valor de k según las tablas como \(P\left(\sqrt{\frac{10}{3}} X \le \sqrt{\frac{10}{3}}k \right) = 0.99\) es decir \(P \left( \text{T}_5 < \sqrt{\frac{10}{3}}k \right) = 0.99\) busco en la tabla añadir diagrama \(P(T_n < a) = \alpha\) y encuentro el valor \(k = \sqrt{\frac{3}{10}} 3.6245 = \dots\)
