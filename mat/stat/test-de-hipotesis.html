Test Aleatorizados

H0 : p = 0.3 vs H1 : p = 0.05 m.a.s n = 5 \( RC = \left\{ \vec{x} \left| \sum x_{i} \ge k \right. \right\} \) \( \alpha = 0.10 = P (\sum x_i \ge k | H_0) \) Pero el problema es que dada nuestra tabla el valor de interés estaría entre el 3 y el 4! pero no hay un valor de ℕ exácto que nos lo de.

En estas condiciones el constraste se realiza de la siguiente manera:

  • Si \( \sum x_{i} < 3 \) se acepta la hipótesis nula.
  • Si \( \sum x_{i} > 3 \) se rechaza la hipótesis nula.
  • Si \( \sum x_{i} = 3\) se lanza una moneda con una probabibildad γ que verifica

Ejemplo continuación

\begin{equation*} \phi(x) = \begin{cases} 1 & \text{si } \sum_{i = 1}^{5} x_i > 3 \\ \gamma = 0.0523 & \text{si } \sum_{i = 1}^{5} x_i = 3 \\ 0 & \text{si } \sum_{i = 1}^{5} x_i < 3 \end{cases} \end{equation*} \begin{align*} \mathbb{E} [\![ \phi(x) ]\!] &= 1 \cdot P(\phi(x) = 1) + \gamma \not P(\phi(x) = \gamma) + 0 \cdot P(\phi(x) = 0) \\ & = P ( \vec{x} \in W_n ) + \gamma P (\vec{x} \in \partial W_n ) \\ \mathbb{E} [\![ \phi(x) | H_0]\!] &= P(\vec{x} \in W_n | H_0 ) + \gamma \cdot P(\vec{x} \in \partial W_n | H_0 )\\ & \implies \gamma = \frac{\alpha - P(\vec{x} \in W_n | H_0}{P( \vec{x} \in \partial W_n | H_0} \end{align*}

Si \( \gamma = 0 \) estamos en el caso del test no aleatoreizado El test de Neyman-Pearson se puede aplicar tambien a los aleatorizados

\begin{equation*} \phi(x) = \begin{cases} 1 &\text{si } \frac{L(x;\theta_{0})_{}}{L(x;\theta_{1})} < k \\ \gamma & \text{si } \frac{L(x;\theta_{0})_{}}{L(x;\theta_{1})} = k \\ 0 & \text{si } \frac{L(x;\theta_{0})_{}}{L(x;\theta_{1})} > k \end{cases} \end{equation*}

Ejemplo Exponencial

Sea \( X \sim (\mu, \sigma) \) con σ conocido m.a.s. de tamaño n

\begin{equation*} P = \left\{ f(\vec{x}) = \left( \frac{1}{\sqrt{2\pi} \sigma} \right)^{n} \cdot e ^{- \frac{1}{2\sigma^2} \sum_{i =1}^n (x_{i} - \mu)}| \theta \in \mathbb{R} \right\} \end{equation*}

tiene CVM en \( T(x) = \sum_{i=1} ^nx_i \) ya que si θ1 < θ2 se tiene que \( \frac{f(\vec{x})| \theta_2}{f(\vec{x})| \theta_1} = \left( \frac{\theta_1}{\theta_2} \right)^n ... \)

Ejemplo Uniforme

Sea \( X \sim U(0,\theta) \) m.a.s. de tamaño n

\begin{equation*} P = \left\{ f(\vec{x}) = \frac{1}{\theta^{n}} \cdot I_{(0, \theta)}\max_i x_i | \theta \in \mathbb{R}^+ \right\} \end{equation*}

tiene CVM en \( T(x) = \max_i x_i} \) ya que si θ1 < θ2 se tiene que \( \frac{f(\vec{x})| \theta_2}{f(\vec{x})| \theta_1} = \left( \frac{\theta_1}{\theta_2} \right)^n ... \)
